# 🏎️ Mercedes F1 - Clone Iniciante com React + Vite

Este é um projeto **iniciante** criado por **João Paulo Azevedo** (azevedo.jpf@gmail.com), com o objetivo de estudar e praticar **React + Vite** de forma moderna, rápida e eficiente.

A ideia principal é recriar o visual e alguns comportamentos do site da equipe **Mercedes-AMG Petronas F1 Team**, com componentes funcionais, estilização em SCSS e layout responsivo.

> Este projeto está em constante evolução e serve tanto como laboratório de testes quanto como portfólio.

---

## 🚧 Status do Projeto

✅ Estrutura inicial com React + Vite  
✅ Componente de banner com visual estilizado  
✅ Layout em grid para sessões como "Notícias" e "Temporada"  
🔧 Em desenvolvimento: sistema de rotas, formulários, integração com APIs e mais seções

---

## 🧠 O que estou aprendendo com este projeto

- React com Vite (ambiente rápido de desenvolvimento)
- Componentização e organização de pastas
- Estilização com SCSS
- Grid Layout e técnicas modernas de CSS
- Git/GitHub com boas práticas de versionamento
- Estruturação de projetos para portfólio
